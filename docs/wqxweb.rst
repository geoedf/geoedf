WQXWeb
====================

.. py:class:: GeoEDF.processor.WQXWeb()

   Module for uploading water quality data to WQX through the WQX Web API using WQXWebLib.

   .. py:attribute:: results_file (str,required)

   The file to write the results to, specify a path. 

   .. py:attribute:: import_config_id (int,optional)

   Defaults the config id to 7411 if not specified. 

   .. py:attribute:: worksheet (int,optional)

   Defaults to 5 if not specified.
