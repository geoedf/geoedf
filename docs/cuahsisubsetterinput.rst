CUAHSISubsetterInput
===========

.. py:class:: GeoEDF.connector.input.CUAHSISubsetterInput()

   This accepts a HUC12 ID as input and submits a request to the CUAHSI subsetter to fetch domain data for this HUC12 watershed.
   
   .. py:attribute:: huc12_id (str,required)

   This is the HUC12 watershed ID that the connector will fetch the domain data.
